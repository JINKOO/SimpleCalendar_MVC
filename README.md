## [모각코_Android스터디 2주차 2021년 11월 17일] Calendar 과제

### [ 주제 ]
* 개인 일정 관리 캘린더(달력 라이브러리 사용 금지)

### [ 적용 Design Pattern ]
* MVC패턴 

### [ 사용 기술 스택 ]
* Kotlin
* Android
* SourceTree
* Git

### [ Feature List ]
### 1. 내가 할 수 있는 기능
  * 첫 화면은 현재 달. 현재 날짜 숫자에 강조 표시(초록색)
  * ‘<-‘ 이미지 버튼을 클릭하면 이전 달로 넘어감.
  * ‘->’ 이미지 버튼을 클릭하면 다음 달로 넘어감.
  * 달력의 특정 칸을 선택하면, 해당 칸의 날짜(2021년 11월 21일 일요일 형식)으로 토스트 출력.


### 2. 오래 걸릴 것 같은 기능
 * 달력 라이브러리를 사용하지 않고, 달력 화면에 그리는 기능.
 * 달력 날짜 계산 알고리즘 구현.


### 3. 추후 기능
 * 특정 날을 선택하면, 해당 날에 일정을 작성할 수 있는 기능 추가.
 * 일정이 있는 날에는 해당 일정 제목 노출.
